## 사용 기술

1. React , React Router
2. postCSS

## 실행 방법  

1. `clone` the repository,  

```
$ git clone "https://github.com/minjuice1/wanted-pre-onboarding-fe.git"
```

2. `Install` dependencies,

```
$ yarn install
```

3. `start` the project,

```
$ yarn start
```
